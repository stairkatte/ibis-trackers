# 3D Print Models — README

## Subfolders and contents
- `Buckles/` — clips and buckles used to secure straps.
- `Dock/` — dock for charging trackers.
- `Receiver/` — case for receiver made from board with added WiFi antenna.
- `Soldering Fixture/` — fixture for soldering IMU to pins. More info in [Documentation](../Documentation/tutorial.md).
- `Tracker V1/` — stacked Smol case. 
    Examples of supported IMU: ICM-45686, LSM6DSV, LSM6DSR.
- `Tracker V1 - Mirrored/` — mirrored version of `Tracker V1`.
- `Tracker V2 Chrysialis/` — case for stacked design that uses Chrysalis IMU module.
    Placement of button makes case incompatible with cases for stacked designs that do not use this shield.
